Application not found

This error most often occurs when an application in one of your dashboards is removed from your bookmarks.

What can I do?

Check if this application is still bookmarked.

To check your bookmarks, go to the Main Menu in the Global Toolbar and select "Bookmarks"

If it is not:
Go to Marketplace and re-bookmark it.

If it is:
Contact the application's Technical Point of Contact (PoC) for help.

To find the Technical PoC, go to your bookmarks and locate the application. Then, open the listing's menu and select "Get help" to view the resources that are available.
